\# Security Policy



\## Reporting Security Issues



If you discover a security vulnerability in these tools, please email: dominicmetz@me.com



Please do NOT open a public issue for security vulnerabilities.



\## File Integrity



Always verify SHA-256 checksums before running tools:

\- See SHA256SUMS file in repo

\- Or check releases page for signed checksums



\## Responsible Use



These tools are for:

✅ Educational purposes

✅ Authorized penetration testing

✅ CTF challenges

✅ Security research with permission



❌ Do NOT use for unauthorized access

❌ Do NOT use for malicious purposes



\*\*I take security seriously. Use these tools responsibly.\*\*
